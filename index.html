About the Project

This project is an AI-powered disaster detection and decision support system that analyzes satellite imagery using SSIM and computer vision techniques. It detects damage, evaluates severity, assigns priority levels, and generates alerts to assist disaster response teams.

Problem Statement

Manual analysis of disaster data is slow and inefficient, delaying response. We need automated systems for fast detection and decision-making.

Need

Automation, speed, and intelligent disaster response systems.

Proposed Solution

Uses SSIM to compare satellite images, classify damage severity, assign priorities, and generate actionable alerts.
