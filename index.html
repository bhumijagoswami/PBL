About the Project

ReliefVision is designed as an intelligent disaster analysis system that processes pre- and post-disaster satellite images using Structural Similarity Index (SSIM). The system detects structural changes, estimates damage severity, and generates prioritized alerts to support decision-making during emergency response scenarios.

Problem Statement

Natural disasters such as floods, wildfires, and earthquakes result in large-scale destruction. Traditional analysis of satellite imagery is manual, time-consuming, and prone to delays, which significantly impacts response time and resource allocation.

Need for System

There is a critical need for automated systems that can rapidly analyze satellite data, detect damage, assess severity levels, and provide actionable insights to assist disaster management authorities in making faster and more informed decisions.

Proposed Solution

The proposed system compares pre- and post-event satellite images using SSIM to identify structural changes. It classifies damage into severity levels (Low, Medium, High), assigns priority scores, and generates alerts with recommended actions, transforming raw image analysis into a complete decision-support pipeline.

Alert & Decision System

The system generates intelligent alerts based on detected damage severity. Each alert includes severity classification, priority ranking, and suggested response actions such as immediate rescue operations, monitoring, or low-risk observation.
