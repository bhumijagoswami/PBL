About the Project

This project is an AI-powered disaster detection and decision support system that analyzes satellite imagery using Structural Similarity Index (SSIM) and computer vision techniques. Unlike traditional systems, it not only detects damage but also evaluates severity, assigns priority levels, and generates intelligent alerts to assist disaster response teams.

Problem Statement

Natural disasters such as floods, wildfires, and earthquakes cause large-scale destruction. Manual analysis of satellite images is slow and inefficient, leading to delayed response. There is a need for automated systems that can quickly detect damage, assess severity, and assist in decision-making for faster disaster response.

Proposed Solution

The system compares pre- and post-disaster satellite images using SSIM to detect structural changes. It classifies damage severity (Low, Medium, High), assigns priority levels, and generates actionable alerts with recommended response strategies. The platform supports multiple disaster types including floods, fires, and earthquakes.

Alert & Decision System

The system generates intelligent alerts based on detected damage severity. Each alert includes severity level, priority ranking, and recommended actions such as immediate rescue, monitoring, or low-risk observation. This transforms the system from a detection tool into a decision-support platform.
